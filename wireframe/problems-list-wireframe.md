## Layout Notes:
- Top navigation bar with logo and user actions
- Stats row with three cards: Solved count, Pass rate percentage, Recent activity sparkline
- Search bar and difficulty filter buttons
- Problem list with status indicators (✓ solved, ✗ attempted, ○ not attempted)
- Each problem shows number, status, title, and difficulty badge
- Clickable rows that navigate to individual problem pages
